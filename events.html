JavaScript Events

Events in JavaScript allow you to respond to user actions and interactions with your web page. Here are some common events and how they can be utilized:

  • onchange: Some HTML element has been modified.
  • onclick: An HTML element has been clicked.
  • onmouseover: The mouse pointer is over an HTML element.
  • onmouseout: The mouse pointer leaves an HTML element.
  • onmouseup: A mouse button is released over an HTML element.
  • onkeydown: A key on the keyboard is pressed down.
  • onload: The HTML page has finished loading.

Example: User-Defined Functions and Events

function closeDoc() {
  alert("You are closing this page!");
  window.close();
}

Here's a breakdown:

  • function closeDoc() {: This line declares a function named closeDoc.
  •   alert("You are closing this page!");: Inside the function, an alert is used to display a pop-up with the message "You are closing this page!" when the function is called.
  •   window.close();: Following the alert, the window.close() statement attempts to close the current browser window or tab.
  • }: Closes the function block.
