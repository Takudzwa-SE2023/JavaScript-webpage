JavaScript JQuery

jQuery is a popular JavaScript library that simplifies the process of creating dynamic and interactive web pages. It is designed to make things easier for developers by providing a concise and convenient way to manipulate HTML documents, handle events, and perform various common tasks.

Take this for example:Imagine a web page as a digital document or a virtual interactive sheet of paper. This sheet contains various elements like text, images, buttons, and forms. To make this sheet more dynamic and responsive, we often need to use JavaScript, which is a programming language for web browsers. Now, imagine JavaScript as a set of tools you can use to interact with this virtual sheet - changing text, hiding or showing elements, or responding to button clicks. While JavaScript is powerful, it can be a bit verbose and require writing many lines of code for common tasks. Here's where jQuery comes in. jQuery acts like a helpful assistant that simplifies the process. It provides a collection of pre-built functions that allow you to accomplish common tasks with just a few lines of code. It's like having a set of shortcuts or handy tools that make your work more efficient. For example, if you want to hide a specific element with JavaScript, you might need several lines of code. With jQuery, you can do it in just one line:

Selectors

Selectors in jQuery are used to target and manipulate HTML elements on a web page.
Here are a few examples:

  • Element Selector-$("h1") Selects all h1 elements on the page.
  • ID Selector$("#mainHeading") Selects the element with the ID "mainHeading".
  • Class Selector$(".yellow")Selects all elements with the class "yellow".
  • Universal Selector$("*")Selects all HTML elements on the page.
  • This Selector:$(this) Selects the current HTML element within a jQuery event handle

Creating and appending elements

This involves dynamically generating HTML elements and adding them to the DOM (Document Object Model). This process is commonly used to build or modify the structure of a web page dynamically

// Creating a new paragraph element let newParagraph = $("

").text("This is a new paragraph."); // Appending the new paragraph to the body $("body").append(newParagraph);

Here's a breakdown:

  • // Creating a new paragraph element: This line initializes a new jQuery object representing a paragraph element (<p>) with an empty text content.
  • let newParagraph = $("

  • // Appending the new paragraph to the body: This line appends the newly created paragraph element to the end of the <body> element in the HTML document.
  • $("body").append(newParagraph);: This line uses the jQuery append() method to add the new paragraph to the end of the <body> element.

Common built-in functions in jQuery incude:

  • after() - Inserts content after the selected HTML element.
  • append() - Inserts content at the end of the selected element.
  • before() - Inserts content after the selected HTML element.
  • empty() - Removes the child elements from the selected element but not the element itself.
  • prepend() - Inserts content at the beginning of the selected element.
  • text() - Sets or returns the text content of the selected element.
  • val() - Sets or returns the value of the form field.
  • remove() - removes the selected element, including its child element.

Events and Effects

In jQuery, events and effects are two important concepts that facilitate interactive and dynamic behavior in web pages.

$("p").click(function(){ $("This").hide('slow'); })

Here's a breakdown:

  • $("p"): This line selects all <p> elements in the HTML document.
  • .click(function() { ... }): This line attaches a click event handler to the selected <p> elements. The function inside the parentheses will be executed when any <p> element is clicked.
  • $("This"): This line attempts to select elements with the tag name "This." However, if "This" is meant to be a class, it should be written as $(".This"), and if it's an ID, it should be written as $("#This").
  • .hide('slow'): This line hides the selected element with a slow animation. The string 'slow' indicates a gradual hiding effect.

Common events in jQuery

  • Mouse events: Click, Double click, Mouse enter.
  • Keyboard events: Key press, Keydown, Key up
  • Form events: Submit, Change, Blur
