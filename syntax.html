Basic JavaScript Syntax

The brief overview of the fundamentals of JavaScript syntax and structures will include variables, conditional statements, arrays & loops, maps, built-in functions, and user-defined functions. After each topic, you will be able to try out or practice what has been discussed.

Variables

In programming, a variable is a container or a named storage location that holds a value. It's like a labeled box where you can store information and give it a name. This information could be a number, a piece of text, or any other type of data.

Example: Assigning Values to Variables

Consider the following JavaScript code:

let greeting = "Hello";
// let means the variable can be reassigned
let question = "How are you?";
// const means the variable cannot be reassigned

Here:

  • let greeting = "Hello";: This line declares a variable named greeting and assigns the value "Hello" to it.
  • // let means the variable can be reassigned;: The // indicates a comment. In JavaScript, comments are for adding explanations and are ignored by the computer.
  • let question = "How are you?";: This line declares another variable named question and assigns the value "How are you?" to it.
  • // const means the variable cannot be reassigned;: This comment explains that using const instead of let would mean the variable cannot be reassigned.

Comments are helpful for explaining your code to others (and to your future self!).

Conditional Statements

Conditional statements in programming allow you to make decisions based on certain conditions. These conditions determine the flow of the program, allowing it to execute different blocks of code depending on whether the conditions are true or false.

In JavaScript, the if, else if, and else statements are commonly used for creating conditional logic. The basic syntax is:

if (condition1) {
// Code to execute if condition1 is true
} else if (condition2) {
// Code to execute if condition2 is true
} else {
// Code to execute if none of the conditions are true
}

Here, condition1, condition2, and so on, are expressions that evaluate to either true or false.

Example: Using Conditional Statements

if (greeting === "Hello" && question === "What's the time?") {
console.log("It is 10 o'clock.");
} else if (greeting === "Hello" || question === "What is the weather like tomorrow?") {
console.log("It will be sunny tomorrow.");
} else {
console.log("I don't understand the question.");
}

Arrays and Loops

In programming, an array is like a list of items, and loops are used to go through each item in the list.

Example: Reversing and Displaying an Array

let classMarks = [62, 88, 90, 70, 69, 85];
for (let i = classMarks.length - 1; i >= 0; i--) {
  console.log(classMarks[i]);
}

classMarks.forEach(function (number) {
  console.log(number + 2);
});

let counter = 0;
let j = 0;
while (counter <= 4) {
  console.log(classMarks[j]);
  counter++;
  j++;
}

This code demonstrates different ways to iterate through an array and display its elements:

  • The first for loop reverses the array and outputs each element.
  • The forEach method adds 2 to each element and outputs the result.
  • The while loop outputs the first 5 elements of the array.

    • There are more array methods that can be used, see below:


    • indexOf() - Searches for the given elements and returns its position.
    • includes() - Checks if an array containes a given element.
    • remove() - Removes an item from the list.
    • pop()- Removes the last element in the array
    • push()- Adds an element to the end of the array and returns the new length
    • shift()- Removes the first element in the array
    • sort()- Sorts elements in the array in ascending order
    • splice()- Adds or removes elements in an array
    • reverse()- Reverses the order of elements in the array

    Maps

    Maps are key-value pairs where each key must be unique, and it can hold any data type as a key or value. Let's explore some methods available for Maps:

    let teachers = new Map();
    teachers.set("grade1", "Mrs. Johnson");
    teachers.set("grade2", "Mr. Bennett");
    // Check if a key exists in the map
    console.log(teachers.has("grade1")); // Output: true
    // Get the value associated with a key
    console.log(teachers.get("grade2")); // Output: Mr. Bennett
    // Delete a key-value pair from the map
    teachers.delete("grade1");
    // Display keys
    for (let key of teachers.keys()) {
      console.log(key);
    }
    // Display values
    for (let value of teachers.values()) {
      console.log(value);
    }
    // Display key-value pairs
    for (let [key, value] of teachers.entries()) {
      console.log(`The ${key} teacher is ${value}.`);
    }
    // Clear all entries from the map
    teachers.clear();

    The provided code showcases various methods to work with a Map and understand its key-value pairs:

    • - The has() method checks if a specific key exists in the map, returning a boolean result.
    • - The get() method retrieves the value associated with a given key in the map.
    • - The delete() method removes a key-value pair from the map based on the provided key.
    • - The clear() method removes all key-value pairs, essentially clearing the entire map.

    User-Defined Functions: doubleNumber()

    In programming, user-defined functions allow you to create reusable blocks of code with a specific purpose. These functions take input parameters, perform operations, and return a result. Let's look at an example function called doubleNumber:

    function doubleNumber(number) {
        return number * 2;
    }
    console.log(doubleNumber(10));
    // Outputs 20

    Here's a breakdown:

    • function doubleNumber(number) {: This line declares a function named doubleNumber that takes one parameter, number.
    •     return number * 2;: Inside the function, it multiplies the input number by 2 and returns the result.
    • }: Closes the function block.
    • console.log(doubleNumber(10));: Calls the doubleNumber function with the argument 10 and prints the result to the console.
    • // Outputs 20: This comment indicates that the console output will be 20.

    When you run this code, it will output 20 because doubleNumber(10) returns 10 * 2, which is 20.

    In JavaScript, there are several built-in functions that perform specific tasks. Here are some commonly used string and array functions:

    • charAt() - Returns the character in a string at the given index.
    • indexOf() - Returns the index of the first occurrence of a character in a string.
    • charCodeAt() - Returns the ASCII number of the character at the given index in a string.
    • fromCharCode() - Returns the character of the given ASCII number.
    • replace() - Replaces a matched substring with a new substring in a string.
    • split() - Splits a string into an array of substrings based on a specified delimiter.
    • toUpperCase() - Returns the given string in all uppercase letters.
    • toLowerCase() - Returns the given string in all lowercase letters.
    • join() - Joins all elements of an array into a single string.
    • pow() - Returns the base to the exponent power.
    • min() - Returns the smallest value among elements.
    • max() - Returns the largest value among elements.
    • round() - Returns a number rounded to the nearest integer.

    These functions provide essential functionality for manipulating strings and arrays in various ways.
